# First Year Rerport
Repository containing sample code for the work presented by Constantino de la Vega in his first year report for full admission into the PhD in Psychiatry at the University of Cambridge. 

Explanation of folders:
- LDSC: Initial LDSC analyses for univariate GWASs (Figure 2AB)
- GSEM: Exploratory and confirmatory analyses (GSA, parallel analysis, EFA, and GenomicSEM modelling) (Figure 2CD, Supplementary figure 1, supplementary figure 2, supplementary table 1, figure 3A)
- multivariate_GWAS: Code to replicate multivariate GWAS generation and LDSC analyses with multivariate GWAS factors (figure 3B, table 1)
- SuSiE_coloc: Containing code to create genomic loci, identify overlapping loci, run SuSiE fine-mapping, and SuSiE colocalisation (figure 4)
- FLAMES_sample_code: Containing code and notes on running the FLAMES pipeline as well as to create Figures 5 and 6 
- Enrichr: All code and data necessary to replicate gene-set enrichment analyses and plot figure 7.
